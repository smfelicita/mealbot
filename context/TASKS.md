## 📋 Бэклог (детальный)

### Рецепты
- [ ] Валидация обязательных полей при создании рецепта (фронтенд)
- [ ] Галерея фото: до 10 штук, выбор одного основного
- [ ] Карточка рецепта для неавторизованного: похожие рецепты по ингредиентам
- [ ] Карточка рецепта для авторизованного — 3 блока рекомендаций:
  - Рецепты из продуктов в холодильнике (все ингредиенты есть)
  - Рецепты с 1–3 недостающими ингредиентами ("купите ещё X")
